C) CSS Files and Structure - top

Styles

We're using 6 CSS files in this template. We have separated styles for improving convenience to edit this template. If you're going to use this template in production we recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

font-awesome.css

Base styles for FontAwesome icon fonts.

bootstrap.css

This Stylesheet contains responsive grid system, default element style based on bootstrap. It is bootstrap framework main css file.

flickity.css

Styles for portfolio slider (flickity slider).

jquery.tosrus.all.css

Portfolio popup gallery css file.

default.css

This is default theme color file for this template.

main.css

Common css style and structures for this template.
